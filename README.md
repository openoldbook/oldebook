# OldEbook 公版书库

> 自由分享公版古籍与民国书籍的数字化图书馆

本项目致力于收集、整理、免费分享已进入公有领域的古籍与民国书籍高清扫描版，让更多人能够自由获取这些珍贵的文化遗产。

## 📚 书籍目录

访问 [在线书库](https://openoldbook.github.io/oldebook/) 浏览全部书籍。

或直接在 [books/metadata.json](books/metadata.json) 查看完整索引。

## 📖 关于书籍来源

本书库所有书籍均来自各大图书馆公开数字化资源，包括但不限于：

- 国家图书馆·数字方志
- 上海图书馆·家谱数据库
- 哈佛燕京图书馆·数字化古籍
- 日本国会图书馆·近代数字图书馆
- Internet Archive 公版书籍

书籍扫描件上的馆藏水印为来源机构的事实性标记，本书库尊重各机构的数字化劳动，并在每本书籍页面注明原始来源。

## ⚖️ 版权声明

本书库只收录**已进入公有领域**的书籍：

- **古籍**：作者去世已满 70 年（依据《伯尔尼公约》通用标准）
- **民国书籍**：作者去世已满 70 年，或出版满 70 年且无法确认版权归属

每本书籍均附有版权状态说明，若您认为某书籍不应在此收录，请提交 Issue，我们将在核实后处理。

本书库本身采用 **CC0 1.0 公众领域贡献声明**，书籍元数据与索引可自由使用。

### 关于收录错误的说明

本人能力有限，尽管每本书在收录前均会人工审核版权状态，**但仍有可能因以下原因误收不应收录的书籍**：

- 作者卒年考证有误或信息来源不准确
- 民国时期部分书籍的版权归属复杂，难以逐一核实
- 个别书籍虽已停版，但版权尚未进入公有领域
- 数字化机构本身对扫描件的管理政策发生变化

**若您认为某书籍不应在此收录，请通过以下方式联系我：**

1. 在 [GitHub 提交 Issue](https://github.com/openoldbook/oldebook/issues/new)
2. 加入企业微信交流群直接联系我
3. 发送邮件至：youebook@qq.com

收到申诉后，我将在 **7 个工作日内核实并处理**（下架相关链接）。对于因误收带来的不便，深表歉意。

本书库无任何盈利目的，所有书籍免费分享，如涉版权问题以撤下处理为准，不接受其他索赔。
